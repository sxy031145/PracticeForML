第一个机器学习项目，主要用于熟悉机器学习项目的各个步骤。
首先需要一个Python机器学习开发环境，需要安装SciPy、Numpy、Matplotlib、Pandas等。
一个机器学习项目主要包括导入数据、概述数据、数据可视化、评估算法和实施预测五个步骤。
1.导入数据
首先导入项目中要使用的类库和方法，然后将数据集保存在项目的统计目录中，用Pandas来导入数据和进行描述性的统计分析。为了有助于对数据的展示，可以为每个数据特征设定名称。用read_csv()导入数据集。
2. 概述数据
主要包括以下几个角度：数据的维度、查看数据、统计描述数据特征（中位数、均值、四分位值等）和数据分类的分布情况。
3. 数据可视化
使用单变量图表可以更好理解每一个特征属性，而多变量图表可以用于理解不同特征属性之间的关系。单变量图表包括直方图、箱线图等，多变量图表可以选取散点矩阵图。
4. 评估算法
主要包括以下几个步骤：分离出评估数据集，采用10折交叉验证来评估算法模型，生成若干不同的模型来预测新数据，选择最优模型。这个项目里采用了线性回归（LR）、线性判别分析（LDA）、K近邻（KNN）、分类与回归树（CART）、贝叶斯分类器（NB）和支持向量机（SVM）6种算法。调用cv_results.mean()可以得到各自的准确度得分，从而选择出准确度最高的算法。
5. 实施预测
使用预留的评估数据集来验证这个模型，并可以通过调用classification_report()给出包含若干指标的报告。
